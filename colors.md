# App Color Palette

This document lists all the colors used throughout the application, categorized by their usage.

## Global & Theme Colors

### Base
- **Background**: `#f1f3f5` (Light Gray/White smoke) - Used for global app background.
- **Surface**: `#ffffff` (White) - Used for cards, panels, and content areas (via Tailwind `bg-white`).
- **Text**: `#888888` (Gray) - Used for secondary/footer text.
- **Borders**: `#ced4da` (Gray) - Used for table cell borders.

### UI Elements
- **Table Headers**: `#f1f3f5`
- **Table Selection**: `rgba(200, 200, 255, 0.4)`
- **Column Resize Handle**: `#aad4ff` (Shortened as `#adf`)
- **ReactFlow Background**: `#aaaaaa` (Dots)
